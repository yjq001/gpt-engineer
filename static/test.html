API 代理接口

此接口用于解决跨域问题，可以代理转发 HTTP/HTTPS 请求到任意目标 URL。

端点： /api/proxy/forward

支持方法： GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS

参数：

  • url: 要转发的 URL (需要进行 URL 编码)
  • method: 请求方法 (默认为 GET)
  • 其他参数将作为查询参数传递给目标 URL

特性：

  • 自动禁用 SSL 证书验证
  • 保留原始请求头和请求体
  • 支持所有 HTTP 方法

Google 登录接口

此接口用于处理 Google OAuth 登录认证。

端点： /api/auth/google

方法： POST

请求体：

  • idToken: Google 认证返回的 ID 令牌

响应：

  • 用户信息和 JWT 访问令牌

特性：

  • 验证 Google ID 令牌
  • 自动创建或更新用户信息
  • 生成 JWT 访问令牌
